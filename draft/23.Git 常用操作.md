## <a name="what"></a> Git 是什么？
Git 是一个免费开源的分布式版本控制系统。无论大小项目，它可以高效快速管理版本相关的事情，例如跟踪记录每一个项目文件的变动。比较有意思的是名称“Git”翻译过来是“饭桶，无用的人”的意思。具体为什么这么命令及Git的发展历史，见[Git][url-wiki-git]。
